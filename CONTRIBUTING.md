# Contributing

## Sign-off required

Every commit must include a `Signed-off-by` trailer (DCO):

```
Signed-off-by: Your Name <your@email.com>
```

Pass `-s` to git and it adds it automatically:

```bash
git commit -s -m "your message"
```

By signing off you certify that you wrote the code and have the right to
contribute it under the project license. See the `DCO` file for the full text.

Pull requests without sign-offs will not be merged.
