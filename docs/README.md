# wasmpeg docs

wasmpeg is a WebAssembly build of FFmpeg with a WebGPU-accelerated scale pipeline and a decoder API for frame-by-frame video/image access.

- [Building](building.md) — prerequisites, build commands, incremental rebuilds
- [Testing](testing.md) — running the suite, adding tests, debugging failures
- [Configuration](configuration.md) — codecs, filters, protocols, external libraries
- [API Reference](api.md) — C exports, `gpu` namespace, `FFmpeg` class
